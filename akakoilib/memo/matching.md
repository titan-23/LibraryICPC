| | |
| :-- | :-- |
| **最大独立集合**  | 辺を共有しない頂点の最大集合            |
| **最小頂点被覆**  | 全ての辺を少なくとも1つの端点で被覆する最小集合  |
| **最大マッチング** | 共有頂点を持たない辺の最大集合           |
| **最小辺被覆**   | 全ての頂点を少なくとも1本の辺で被覆する最小辺集合 |

- 最大独立集合 + 最小頂点被覆 = 頂点集合
  - 頂点被覆は独立集合の補集合
  - 頂点被覆の補集合から2頂点を取ってきて隣接していたら、その辺は被覆されないため
- 最小辺被覆 + 最大マッチング = 頂点数 (孤立点を除く)
  - マッチングの端点となっていない頂点につき一本ずつ辺を追加すると辺被覆になる
- 最大マッチングのサイズ = 最小頂点被覆のサイズ (二部グラフ)
- 最小辺被覆のサイズ = 最大独立集合のサイズ (二部グラフ / DAG)
- DAGの最小辺被覆は、各頂点を入る頂点と出る頂点に分けてできる二部グラフにおける最大マッチングのサイズを頂点数から引いた値として求めることができる
